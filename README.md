# Tanks-War-2D-2Player
Immerse yourself in a retro game, get on a tank and blow up your opponent. You need 2 players to play
## No library
This game no containst outside library. It use pure html5 and javascript.
## Control
Drive, shoot, mine
### First player
```
W, A, S, D / .
```
### Second player
```
← → ↑ ↓ G H
```
